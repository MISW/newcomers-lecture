#### 1～40までの偶数のみを表示するプログラム
```
#include<stdio.h>

int main(){
	
	int i;
	
	for(i=2;i<=40;i+=2){
		
		printf("%d\n",i);
		
	}
	
	return 0;
}
```

#### 以下解説
```
#include<stdio.h>

int main(){
	
	int i;
	
	for(i=2;i<=40;i+=2){
		
		printf("%d\n",i);
		
	}
	/*
	まずは初期命令でiに2が代入されます。
	今までループの最終部分の命令でi++のみを用いてましたが、
	i++ではなくi+=2なんてのも可能です。
	この場合iに1ずつではなく 2ずつ足されていくので
	iは2,4,6,8・・・となっていくのがわかるでしょう。
	つまり偶数しか表示されないということです。
	*/
	
	return 0;
}
```
#### また逆に40～1までの偶数（つまり40 38・・・2)を表示するプログラム
```
#include<stdio.h>

int main(){
	
	int i;
	
	for(i=40;i>0;i-=2){
		printf("%d\n",i);
	}

	/*
	初期命令でi=40
	i>0ならばループが継続され,
	ループの最終部分でi-=2よりiが2ずつ減っていきます。
	*/
	
	return 0;
}
```

#### まとめ
ループの最終命令部分ではi++,i--以外にもいろいろ使用可能
