#### 1問目　5の7乗を計算するプログラム
```
#include<stdio.h>

int main(){
	
	int ans,i;
	
	ans=1;
	
	for(i=0;i<7;i++){
		
		ans *= 5;
		
	}
	
	printf("5^7 = %d\n",ans);

}
```
for(i=0;i<=6;i++)でも可

#### 2問目  9の階乗を計算するプログラム
```
#include<stdio.h>

int main(){
	
	int ans,i;
	
	ans = 1;
	
	for(i=1;i<=9;i++){
		
		ans *= i;
		
	}
	
	printf("9! = %d\n",ans);
	
	return 0;
}
```
for(i=1;i<10;i++)でも可
