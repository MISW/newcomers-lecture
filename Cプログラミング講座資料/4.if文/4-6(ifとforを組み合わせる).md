#### 問題
x+=2を用いずに1～200までの偶数を表示してください。

#### 以下解答
```
#include<stdio.h>

int main(){
	
	int i;
	//ループを回すためのi	

	for(i=1;i<=200;i++){//今回は1～200までの数が欲しいのでこのように設定
		if(i%2 == 0){//2で割り切れたら
			printf("%d\n",i);
		}
	}
	
	return 0;
}
```
以前作った偶数を表示するプログラムですが、この様にすることでも偶数を表示することが可能です。
