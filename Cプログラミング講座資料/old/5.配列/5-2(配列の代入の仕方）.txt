#include<stdio.h>

int main(){
	
	int num[3]={3,20,2};
	
	printf("%d %d %d\n",num[0],num[1],num[2]);
	
	return 0;
}


/*
以下解説

配列は宣言と同時に代入することも可能。
代入する時は

型名　変数名[要素数] = {(0個目の変数に代入する数) , (1個目) , (2個目), ・・・ };

とする。

なのでこの時、
num[0] = 3;
num[1] = 20;
num[2] = 2;

となるのと、同様である。
実際にprintfで出力できるので確認できると思われる。

*/
