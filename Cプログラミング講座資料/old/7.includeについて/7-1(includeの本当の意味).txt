#include<stdio.h>
#include<math.h>

int main(){
	
	double a,b;
	
	a = sin(0);
	b = cos(0);
	
	printf("%f %f\n",a,b);
	
	return 0;
}

/*
以下解説

今回今までと違うのが
#include<math.h>
というのが追加されたことである。

この〜〜.hというのはいろいろな命令が書いてあるものであり、
includeというのは、その命令を使いますよーという宣言なのである。

そこで今回追加したmath.hというのは数学関数とよく呼ばれており、
みんなおなじみの三角関数や、逆三角関数、平方根などが入っている。

今回はsinとcos関数を使って、数値を表示してみた。
sinとcosが取る値を変えてみてやってみてもよいだろう。

*/
